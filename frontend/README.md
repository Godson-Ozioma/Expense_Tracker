# Expense Tracker — frontend

React + Vite SPA for [Expense Tracker](../README.md).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Dev server at `http://localhost:5173` with API proxy to `http://127.0.0.1:8000` |
| `npm run build` | Production build → `dist/` |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | ESLint |

Environment variables are documented in **`.env.example`** and in the root **README**.
